# CyberGuard SaaS

CyberGuard is a modern Streamlit-based cybersecurity web app that reviews website defensive configurations and generates client-ready PDF reports.

## Features
- Login and signup
- Website security review
- HTTPS and TLS checks
- Security header analysis
- Cookie flag checks
- PDF report export

## Run locally
```bash
pip install -r requirements.txt
streamlit run cyberguard_streamlit_app.py
